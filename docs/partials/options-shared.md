### `verbose`

Use verbose output.

- **CLI option**: `-v,--verbose`
- **JS key**: `verbose`
- **Type**: `boolean`

### `time`

Print execution time.

- **CLI option**: `-t,--time`
- **JS key**: `time`
- **Type**: `boolean`

### `exit`

Force exit from process on error.

- **CLI option**: `-e,--exit`
- **JS key**: `exit`
- **Type**: `boolean`
